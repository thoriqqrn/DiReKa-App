**Rumus perhitungan kebutuhan :** 

- **BBI			: (TB-100)-10%**  
- **BMR laki laki 		: (10 x BBI) \+ (6,25 x TB) \-(5 \- u) \+5**  
- **BMR perempuan 	:  (10 x BBI) \+ (6,25 x TB) \-(5 \- u) \-161**  
- **Total energi 		: BMR x Skor faktor aktivitas x 1**  
- **Karbohidrat 		: 55% x total energi**   
- **Protein 		: 20% x total energi**  
- **Lemak 		: 25% x total energi**   
- **Natrium 		: 2.300 mg/hari jika tekanan darah 140-159/90-99mmHg**  
  			**1.500 mg/hari jika tekanan darah \>160/100**  
- **Kalsium 		: 1200 mg/hari**  
- **Magnesium 		: 350 mg/hari**  
- **Kalium		: 4.700 mg/hari**
